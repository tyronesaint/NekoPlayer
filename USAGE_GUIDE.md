## 🚀 快速开始

### 首次使用

1. **打开应用**
   - 首次启动时，您会看到 NekoPlayer 的欢迎界面
   - 主界面包含三个标签：文件、播放、设置

2. **添加视频文件**
   - 点击"文件"标签
   - 选择添加本地文件或连接 WebDAV 服务器

3. **开始播放**
   - 从列表中选择视频文件
   - 播放器会自动加载并开始播放

### 本地文件播放

1. 点击"文件"标签
2. 点击"添加本地文件"
3. 从文件选择器中选择视频文件
4. 支持格式：
   - MP4 (推荐)
   - MOV
   - M4V
   - AVI
   - MKV

## ▶️ 播放控制

### 基本控制

- **播放/暂停**: 点击屏幕中央或使用播放/暂停按钮
- **前进/后退**: 使用时间轴拖拽或点击前进/后退按钮
- **音量**: 使用设备音量键或播放器界面音量滑块

### 进度控制

- **快速跳转**: 双击屏幕左侧/右侧快速后退/前进 10 秒
- **精确跳转**: 拖动进度条到指定位置
- **标记位置**: 长按进度条设置书签

### 播放列表

- **列表视图**: 点击列表按钮查看所有文件
- **顺序播放**: 自动播放下一个文件
- **循环播放**: 设置中开启循环模式

## 🎨 Anime4K 增强设置

### Anime4K 模式

NekoPlayer 支持多种 Anime4K 增强模式：

1. **A (AA-Mode)**
   - 标准 Anime4K 模式
   - 适合大多数动画
   - 性能平衡

2. **B (AA-Mode + Bilibili)**
   - 结合 Bilibili 增强
   - 更强的线条恢复
   - 推荐用于新番

3. **C (A-Average + Bilibili)**
   - 平均化处理
   - 适合高压缩源

4. **D (Strength Mode)**
   - 强化模式
   - 适合低质量源

5. **Fast**
   - 快速模式
   - 性能优先
   - 适合实时播放

### 使用建议

| 源质量 | 推荐模式 | 预期效果 |
|--------|----------|----------|
| 高质量 (1080p+) | A 或 Fast | 细节增强 |
| 中等质量 (720p-1080p) | B 或 C | 线条恢复 |
| 低质量 (<720p) | D 或 C | 显著提升 |
| 实时流媒体 | Fast | 流畅优先 |

### 性能优化

- **降低预览质量**: 优先保证播放流畅
- **关闭音频**: 仅增强视频画面
- **调整模式**: 使用 Fast 模式获得最佳性能

## 🌐 WebDAV 配置

### 添加 WebDAV 服务器

1. **进入设置**
   - 点击"设置"标签
   - 选择"WebDAV 服务器"

2. **添加新服务器**
   - 点击"添加服务器"按钮
   - 填写服务器信息：
     - **名称**: 服务器昵称（如"我的 NAS"）
     - **URL**: WebDAV 服务器地址
     - **用户名**: 可选（如果需要认证）
     - **密码**: 可选（如果需要认证）

3. **连接服务器**
   - 点击已保存的服务器
   - 浏览文件列表
   - 选择视频文件播放

### WebDAV 服务器示例

#### Synology NAS
```
URL: https://your-nas-ip:5006/webdav/
用户名: your-username
密码: your-password
```

#### Nextcloud
```
URL: https://your-nextcloud-domain/remote.php/webdav/
用户名: your-email
密码: app-password
```

#### ownCloud
```
URL: https://your-owncloud-domain/remote.php/webdav/
用户名: your-username
密码: your-password
```

### WebDAV 使用技巧

- **预加载**: 在 Wi-Fi 环境下预加载视频
- **缓存**: 已播放的视频会被缓存到本地
- **断点续传**: 支持从上次播放位置继续

## ⚙️ 高级功能

### 播放列表管理

1. **创建播放列表**
   - 点击"播放列表"按钮
   - 点击"新建列表"
   - 输入列表名称
   - 添加文件到列表

2. **导入/导出**
   - 设置中选择"播放列表"
   - 导出为 JSON 格式
   - 支持备份和分享

### 快捷键 (macOS)

- **空格**: 播放/暂停
- **⬅️/➡️**: 前进/后退 5 秒
- **⬆️/⬇️**: 调整音量
- **F**: 全屏切换
- **M**: 静音
- **⇧ + ⬅️/➡️**: 前进/后退 30 秒

### 边播边录

- **录制截图**: 点击截图按钮保存当前帧
- **录制片段**: 点击录制按钮开始/停止
- **保存位置**: 照片应用

### 字幕支持

- **自动加载**: 与视频同名的 SRT/ASS 字幕
- **手动加载**: 设置中选择"字幕" → "添加字幕"
- **样式调整**: 字体、大小、颜色、位置

### 音频轨道选择

- **多音轨**: 设置中选择"音频" → 选择音轨
- **音效增强**: 开启虚拟环绕声

## ❓ 常见问题

### 播放问题

**Q: 视频无法播放？**
A:
1. 检查文件格式是否支持
2. 确认文件未损坏
3. 尝试降低分辨率
4. 关闭 Anime4K 模式测试

**Q: 播放卡顿？**
A:
1. 使用 Fast 模式
2. 关闭预览
3. 降低分辨率
4. 关闭后台应用

**Q: 画面花屏？**
A:
1. 切换 Anime4K 模式
2. 更换视频解码方式
3. 清除缓存重新播放

### Anime4K 问题

**Q: Anime4K 效果不明显？**
A:
1. 确认源质量（低质量源效果更明显）
2. 尝试不同的模式
3. 调整强度参数

**Q: Anime4K 导致性能问题？**
A:
1. 使用 Fast 模式
2. 降低预览质量
3. 关闭音频增强

### WebDAV 问题

**Q: 无法连接 WebDAV 服务器？**
A:
1. 检查网络连接
2. 确认 URL 正确（以 / 结尾）
3. 验证用户名密码
4. 检查服务器 WebDAV 功能是否开启

**Q: WebDAV 播放很慢？**
A:
1. 在 Wi-Fi 环境下使用
2. 降低视频质量
3. 使用缓存功能

### 其他问题

**Q: 如何更新应用？**
A:
- 从 GitHub 下载最新版本
- 覆盖安装即可

**Q: 支持哪些设备？**
A:
- iOS 15.0+
- macOS 12.0+
- tvOS 15.0+

**Q: 数据安全吗？**
A:

- WebDAV 凭证本地加密存储
